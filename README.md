# compact-disc-collective
A digital music archive and portfolio project built with Vue 3 and Tailwind CSS. The application fetches album data and presents them as interactive, physical CD cases where the disc smoothly slides out and plays real-time audio previews upon interaction.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
